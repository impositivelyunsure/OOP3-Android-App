Android app developed in Android studio using Kotlin & Python.

LLM emotion analysis and short feedback for user inputted journal entries. Uses FastAPI written in Python to handle send/receives structured in JSON.
Analysed emotions are displayed in a pie chart, associated journal entries and feedback is stored below it inside an interactive list. Emotions can be searched using a binary tree algorithn, hashmap, and doubly linked list.
Sorting the emotions can be done by using a bubble sort, insertion sort and selection sort.
