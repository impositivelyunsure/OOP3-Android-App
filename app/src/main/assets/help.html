Help

Write a journal entry and tap Analyze. The app sends your text to the local FastAPI server that uses Ollama gemma:2b to detect the dominant emotion and return a one-line advice.

Features

  • Emotion + advice card
  • History list (drag an item onto the Trash to delete)
  • Sort by emotion (Bubble / Insertion / Selection)
  • Search by emotion (HashMap / Binary Tree / Doubly Linked List)
  • Pie chart of emotion distribution

Troubleshooting

  • Ensure FastAPI is running: uvicorn main:app --host 0.0.0.0 --port 8000 --reload
  • Run Ollama and have gemma:2b pulled: ollama serve, ollama pull gemma:2b
